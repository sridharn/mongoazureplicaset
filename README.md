## MongoDB Replica Sets on Azure ## 

## Building
### Prerequisites
  * .Net 4.0.
  * Visual Studio 2010 SP1
  * Windows Azure SDK 1.5 
  * Windows Azure Tools for Visual Studio 2010 1.5

### Build
  * Open MongoAzure.sln from Visual Studio 2010 and build
